# EcoBreeze Website

## UAE's #1 Eco-Friendly Pest Control & Deep Cleaning Website

A complete, professional website for EcoBreeze - built with all Urban Company features.

### Features
- **Full Service Booking System** - Multi-step booking modal with service selection
- **Live Search** - Urban Company-style instant service search with dropdown
- **Location Selector** - Dubai, Abu Dhabi, Sharjah, Ajman, RAK, Fujairah, UAQ
- **Hero Section** - Animated booking widget with 6 service cards
- **Service Categories** - 8 category cards (Urban Company grid style)
- **Deep Cleaning Services** - 8 services with pricing
- **Pest Control Services** - 6 services with pricing  
- **Packages Section** - 3 tiers (Home, Villa, Business Pro)
- **How It Works** - 4-step process
- **Expert Technicians** - Profile cards with ratings and booking
- **Live Tracking UI** - Phone mockup showing real-time tracking
- **Before/After Gallery** - 5-card photo gallery
- **Reviews Slider** - 6 reviews with pagination
- **Service Areas** - All 7 UAE Emirates
- **FAQ Accordion** - 8 common questions
- **Contact Form** - Full contact section
- **App Download Section** - App Store & Google Play
- **My Bookings Page** - Full booking management UI
- **Login/Signup Modals** - Account management
- **Floating Action Buttons** - WhatsApp, Call, Book Now
- **Mobile Responsive** - Full responsive design

### Files
- `index.html` - Main homepage
- `styles.css` - Complete CSS styles
- `scripts.js` - Interactive JavaScript
- `mybookings.html` - My Bookings management page

### Quick Start
Open `index.html` in any browser or deploy to any web host.

### Customization
Replace `+971 50 XXX XXXX` with real phone number.
Replace `info@ecobreeze.ae` with real email if needed.
Replace Trade License number with real one.
Add real images to gallery section.
